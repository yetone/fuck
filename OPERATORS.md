# Operators

| Arithmetic operators  | Syntax          |
| :------------         | :----           |
| Addition              | a + b           |
| Subtraction           | a - b           |
| Multiplication        | a * b           |
| Division              | a / b           |
| Modulo                | a % b           |

| Comparison/relational operators   | Syntax          |
| :------------                     | :----           |
| Equal to                          | a == b          |
| Equal to                          | a == b          |
